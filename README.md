## ThreeJS 
###项目安装
```javascript
git clone https://github.com/xuff78/three-example.git
cd three-example
npm i
npm run dev
```
#####基于 ThreeJS + Webpack5，写的几个例子，会持续更新，欢迎关注。

Heatmap：全国地图是由地理坐标转换成空间数据绘制的，配上热力信息，通过heatmap.min.js生成热力图，再使用Three计算点高度后贴图。加了几个飞线的小动画，支持鼠标事件。
